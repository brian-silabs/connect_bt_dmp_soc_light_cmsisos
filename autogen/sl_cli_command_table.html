CLI Command Reference

ble_hello connect_ble_cli_hello_command
Issue an Hello command to the BLE stack
ble_get_address connect_ble_cli_get_address_command
Get the BLE address
ble_set_adv_params u16 u16 connect_ble_cli_set_adv_params_command
Set the BLE advertisement parameters
Arguments
  • u16Minimum interval
  • u16Maximum interval
ble_start_adv u8 u8 connect_ble_cli_start_adv_command
Start BLE advertising
Arguments
  • u8Discoverable mode
  • u8Connectable mode
ble_open_connection hex u8 connect_ble_cli_open_connection_command
Open a BLE connection
Arguments
  • hexAddress
  • u8Address type
ble_close_connection u8 connect_ble_cli_close_connection_command
Close a BLE connection
Arguments
  • u8Connection handle
ble_set_connection_params u16 u16 u16 u16 connect_ble_cli_set_connection_params_command
Set the BLE default connection parameters
Arguments
  • u16Minimum interval
  • u16Maximum interval
  • u16Slave latency
  • u16Supervision timeout
start_tx_stream u8 u16 start_tx_stream_command
Start RAIL TX stream mode
Arguments
  • u8mode. 0 = PN9, 1 = CW
  • u16Channel
stop_tx_stream stop_tx_stream_command
Stop RAIL TX stream mode
form cli_form
Forms a network with a default address
leave cli_leave
Forget the current network and revert to EMBER_NO_NETWORK
reset cli_reset
Reset the hardware
info cli_info
MCU ID, Network state, Node ID, PAN ID, Channel ID, etc.
set_pan_id u16 cli_set_pan_id
Set the PAN ID of the node
Arguments
  • u162 byte PAN ID
set_channel u16 cli_set_channel
Set the communication channel
Arguments
  • u162 byte channel number
set_tx_options u8 cli_set_tx_option
Set Tx options
Arguments
  • u8Security(0x01) MAC Ack(0x02) High Prio(0x04)
set_key hex cli_set_security_key
Set security key
Arguments
  • hexSecurity key (size:EMBER_ENCRYPTION_KEY_SIZE)
unset_key cli_unset_security_key
Unset current security key
